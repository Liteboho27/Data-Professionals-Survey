# Data-Professional-Survey

### 📋 Project Overview

This interactive Power BI dashboard analyses the 2022 Data Professional Survey dataset collected by Alex The Analyst. The report explores key aspects of the data industry, including salary trends by job title, preferred programming languages across different data roles, career satisfaction levels (salary, work/life balance, skills development and upward mobility) and insights into breaking into data careers.

Through extensive data cleaning in Power Query (including splitting multi-value fields like favorite programming languages), DAX calculations, and thoughtful visualisations, the dashboard delivers clear, actionable insights into compensation, skills, and job satisfaction in the data field. As an aspiring Data Analyst passionate about turning raw data into meaningful business stories, this project allowed me to strengthen my skills in end-to-end analytics, from data transformation and modeling to building interactive, user-friendly dashboards that drive better decision-making.

## 🎯 Key Insights

- Data Scientists and Data Engineers have significantly higher salaries compared to Data Analysts.
- Python dominates across all data professions, followed closely by other languages/tools (including SQL)
- Most repondents (42.7%) find it neither difficult nor easy to break into the data profession, but a significant number (24.7%) consider it difficult to break into the profession. The most prominent roles to break into are Data Scientist and Data Architect.
- Better Salary, Remote work and Good work/life balance are the **top 3** most important factors that people conmsider when applying for data roles


## 🛠️ Technologies & Skills Demonstrated

- **Power BI Desktop** – Report design and interactivity
- **Power Query** – Data cleaning, column splitting (multi-value languages), type handling, and transformation
- **DAX** – Calculated measures for averages
- Interactive visuals: Clustered charts, cards, guaging and hierarchy slicers
- Data modeling and relationship management


## 📊 Dashboard Highlights

**Salary Analysis** 
- The **Top 3** earners are Data Scientists, Data Architects and Data Engineers respectively
- In comparison, Data analysts rank 5th

**Skills & Tools** 
- Majority of data professionals prefer using Pyhton in their roles. This makes Python the most relevant programming language across multiple data roles
- The Skills development rating is above average (5.61), showing an overall satisfaction with learning new skills and knowledge in data roles

  
**Satisfaction Metrics**
-  Happiness score on work/life is the highest (5.74) compared to salary (4.27) and upward mobility (4.76). 
  
- **Demographics** – Gender and age distribution by job title with drill-down capability
- **Career Entry** – Difficulty breaking into data and most important job factors
