# Data-Professional-Survey

### Project Overview

This interactive Power BI dashboard analyses the 2022 Data Professional Survey dataset collected by Alex The Analyst. The report explores key aspects of the data industry, including salary trends by job title, preferred programming languages across different data roles, career satisfaction levels (salary, work/life balance, skills development and upward mobility) and insights into breaking into data careers.

Through extensive data cleaning in Power Query (including splitting multi-value fields like favorite programming languages), DAX calculations, and thoughtful visualisations, the dashboard delivers clear, actionable insights into compensation, skills, and job satisfaction in the data field. As an aspiring Data Analyst passionate about turning raw data into meaningful business stories, this project allowed me to strengthen my skills in end-to-end analytics, from data transformation and modeling to building interactive, user-friendly dashboards that drive better decision-making.
